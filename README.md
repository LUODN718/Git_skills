# Git Daily — Git 每日助手

给 Codex Agent 用的 Git 技能包。用日常对话的方式完成 Git 操作，像和同事说话一样下指令，不用记命令。

## 安装

将 `git-daily/` 文件夹放入 `~/.codex/skills/` 目录，重启 Codex 即可。

## 使用方式

两种触发方式：

**显式调用**：`$git-daily 帮我提交今天的代码`

**自然触发**：直接说"帮我提交一下 / push 上去 / 看看远程有啥更新 / 切到 xxx 分支"，只要包含 Git 相关词汇就会自动激活

## 覆盖场景

| 场景 | 你说 |
|------|------|
| 初始化绑定 | 我刚建了个 GitHub 仓库，帮我绑一下 |
| 日常提交 | 帮我把今天写的提交一下 |
| 协作同步 | 检查一下远程有什么更新 |
| 分支管理 | 新建一个 feature/xxx 分支 |
| 合并冲突 | 把 feature/xxx 合并到 main |
| 回退撤销 | 刚才那次提交不对，帮我撤销 |
| 暂存切换 | 先暂存我的改动，切去修个 bug |
| README 生成 | 帮我总结一下项目，写个 README |
| 查看历史 | 看看最近 5 次提交 |
| 标签管理 | 打个 v1.0 的标签 |
| cherry-pick | 把那个提交摘过来 |
| gitignore | 帮我生成一个 .gitignore |

## 设计原则

1. **口语优先**：用户像跟同事说话一样下指令，Agent 翻译成命令
2. **先看后动**：任何修改操作前先检查状态
3. **安全默认**：不擅自 --force、--hard、删未合并分支
4. **主动确认**：缺少信息时用自然语言询问

## 依赖

无外部依赖。纯指令驱动的 Skill，Agent 使用系统自带的 Git 命令。
